## Tipo informal del USDT a boliviano en Binance P2P

- `vwap`: Media ponderada de ofertas (_un valor justo_)
- `naive`: Valor más frecuente en el 10% extremo de ofertas (_el valor que la gente usa_)
- `median`: Mediana de ofertas

Para [compra](datos/binance/compra.csv) y [venta](datos/binance/venta.csv). Y datos adicionales para comprender tamaño y concentración del mercado en [compra](datos/binance/compra_mercado.csv) y [venta](datos/binance/venta_mercado.csv).
